# 课堂喊人传话系统 v2.0

> 老师用自己的电脑/手机 → 喊人/传话 → 教室公用电脑大屏显示 + 语音播报

## 使用场景

```
老师电脑/手机 ──→ 教室公用电脑（接大屏/投影）
  (控制台)            (显示屏，大字+语音)
```

- 教室里有一台公用电脑（可以接投影仪/大屏）
- 老师在自己电脑或手机上操作，喊人/传话
- 公用电脑大屏显示信息，语音播报学生名字
- 学生看到/听到后回应

## 快速开始

### 1. 启动服务器

双击 `start.bat`，或在终端执行：

```bash
cd classroom-call
venv\Scripts\activate
python server.py           # 默认 9000 端口
python server.py 8080      # 自定义端口
```

### 2. 设置教室显示屏

在教室公用电脑上打开浏览器，访问终端显示的地址：

```
http://192.168.x.x:9000/display
```

然后点左上角「⛶ 全屏」按钮，把浏览器全屏。

**建议：** 把这个地址加入浏览器书签，设为首页，开机自动打开。

### 3. 老师操作

在自己电脑或手机浏览器打开：

```
http://192.168.x.x:9000/teacher
```

操作：
- **喊人**：输入学生姓名 → 点「🔔 喊人」→ 公用电脑大字显示 + 语音播报
- **快速喊人**：点预设的「班长」「课代表」等快捷按钮
- **传话**：输入文字 → 点「💬 发送」→ 公用电脑显示文字
- **关闭通知**：点「✕ 关闭大屏通知」

## 功能说明

| 操作 | 老师端 | 公用电脑显示 |
|------|--------|------------|
| 喊人 | 输入姓名+可选附言 | 🔴 红色全屏 + 大字 + 铃声 + 语音播报 |
| 传话 | 输入文字 | 🔵 蓝色全屏 + 文字 + 语音播报 |
| 关闭 | 点关闭按钮 | 退回待机画面 |

- 喊人通知 30 秒后自动关闭，传话 20 秒后自动关闭
- 底部有消息历史条，可以回顾最近的消息

## 端口说明

默认端口 **9000**。如果被占用，修改 `start.bat` 中的 `PORT=9000` 为其他数字，或命令行传参：

```bash
python server.py 8080
```

## 注意事项

- 确保老师电脑和教室公用电脑在**同一局域网**
- 如果学生电脑无法访问，检查 Windows 防火墙是否允许 Python 通过
- 老师端页面做了手机适配，用手机浏览器也能操作
- 建议教室公用电脑用 Chrome，语音播报效果最好

## 技术栈

- 后端：Python + FastAPI + WebSocket
- 前端：纯 HTML/CSS/JS，零依赖
- 实时通信：WebSocket
- 语音：浏览器原生 Web Speech API
- 音效：Web Audio API 合成
